# PiMaster

A standard Vite + React + TypeScript project with Tailwind CSS.

## Getting started

Install dependencies and start the development server:

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Preview

```sh
npm run preview
```

## Project structure

- `src/` — React application source
- `src/components/` — reusable UI components
- `src/pages/` — route pages
- `src/lib/` — application logic and helpers
- `src/hooks/` — custom React hooks

## Notes

This repository is a standard Vite/React codebase. If you regenerate dependencies, remove or recreate lock files as needed.
